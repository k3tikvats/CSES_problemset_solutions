# CSES Problem Set (C++)

My Solutions to CSES problems, organized by topic. Written and tested in Visual Studio Code on Windows.

- Language: C++17
- IDE: VS Code
- Build: g++ (MinGW)
- Modulo: 1e9+7

## Structure
- dp/ — dynamic programming
- graphs/ — BFS/DFS/Dijkstra, cycles, routes
- sorting_and_searching/ — two pointers, greedy, Kadane
- introductory/, tree/, geometry/ — other topics
